# Release Note: 2026-04-24

这份发布说明记录当前这轮 Windows / WSL 量化研究基线。

## 推荐项目名

- 中文名：`星桥量化`
- 英文名：`StarBridge Quant`
- 保留代号：`starbridge-quant`

采用这个命名的原因：

1. `Bridge` 还能准确表达 Windows QMT 服务端和 WSL 研究端之间的桥接关系。
2. `Quant` 比单纯的 `Bridge` 更能体现现在这套仓库已经覆盖到因子研究、快照、差异审计和量化开发。
3. 对外显示名、分发名和主 CLI 已迁移到 StarBridge 命名，同时保留旧 `qmt-*` 入口兼容，兼顾迁移成本和现网脚本稳定性。

## 本轮基线内容

### Windows 侧

- `starbridge-server` 启动 / 停止 / 健康检查脚本整理
- WSL `18888` 转发与开机启动配套脚本整理

### WSL 侧

- 原生研究虚拟环境迁移到 WSL 本地目录

- 量化数据专员 / 量化因子专员角色文件建立
- QMT `GICS4` 行业映射缓存与行业中性研究模板建立
